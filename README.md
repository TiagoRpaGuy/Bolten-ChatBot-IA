# Proposal Configurator - White Label Sales Tool

Sistema de configuração de propostas comerciais para parceiros de agência, com precificação baseada em valor e análise de ROI.

## Features

- ✅ **Precificação Dinâmica**: Slider de margem + fatores de complexidade
- ✅ **Value-Based Pricing**: Sugestão de preço baseado em 10% do ROI recuperado
- ✅ **J-Curve ROI Timeline**: Visualização do payback em 12 meses
- ✅ **Customização Técnica**: Domínio personalizado e White Label
- ✅ **6 Faixas de Usuários**: Sincronia bidirecional com planos
- ✅ **Tutorial Integrado**: Onboarding para parceiros
- ✅ **PDF Otimizado**: Layout impecável para impressão
- ✅ **Integração Stripe**: Link de checkout configurável

## Instalação

```bash
npm install
npm run dev
```

## Uso

1. Configure o cliente e plano na aba "Config"
2. Use a calculadora ROI para provar valor
3. Ajuste margem e fatores de complexidade
4. Gere o link e compartilhe com o cliente

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
